# GoFast Landing Page

## What is this?

This is the **static HTML landing page** for GoFast. It's a simple marketing site that sits at the root domain and directs users to download the GoFast PR app on the App Store.

## Structure

- **`index.html`** - Main landing page (the one people see when they visit the site)
- **`benefits.html`** - Benefits of training with others
- **`how-it-works.html`** - How GoFast works: goals, plans, training, run discovery
- **`about.html`** - About GoFast
- **`privacy-policy/index.html`** - Privacy Policy (canonical URL: `/privacy-policy/`)

## Purpose

The landing page:
1. Answers "Why should I download GoFast?"
2. Highlights training plans, run discovery, and training with others
3. Explains how to set a goal, build a plan, train, and join runs
4. Provides a CTA to download the app on the App Store

## Deployment

Deployed to Vercel at: `gofastcrushgoals.com`

## Tech Stack

- Static HTML
- Tailwind CSS (via CDN)
- Vite for building

## Notes

- This is NOT the main app - that's in `gofastapp-mvp` / `GoFast-mobile`
- Keep it simple - just a marketing landing page focused on training + community
- Should be fast, minimal, and convert visitors to App Store downloads
