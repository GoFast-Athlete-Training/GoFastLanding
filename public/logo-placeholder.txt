# Logo Placeholder

Drop your logo.jpg file here!

## Instructions:
1. Add your logo.jpg file to this public folder
2. The logo will be accessible at /logo.jpg in the built site
3. Make sure the logo is optimized for web (recommended: 200x200px or larger)

## Current Usage:
- Home page: src/pages/home.html references the logo
- Favicon: index.html uses the logo as favicon
- All pages can access the logo via /logo.jpg

## Supported Formats:
- .jpg / .jpeg
- .png
- .svg (recommended for scalability)
- .webp (modern format, great compression)
